# 🍽️ Cloud Kitchen Manager

A responsive and user-centric frontend web application designed to simulate a cloud kitchen management system. The application allows efficient menu handling, real-time order processing, and dynamic bill calculation, focusing on clean UI and practical usability.

---

## 🎯 Purpose

This project was developed to strengthen frontend development skills, enhance logical problem-solving, and build a practical real-world application suitable for academic evaluation and technical portfolios.

---

## 🚀 Live Demo

👉 https://jahanvisingh07.github.io/cloud-kitchen-manager/

---

## ✨ Key Features

• Add and manage menu items (name and price)  
• Dynamic menu display with real-time updates  
• Order selection with quantity input  
• Automatic bill calculation based on selected items  
• Clear order functionality for resetting transactions  
• Persistent data storage using browser localStorage  
• Clean, simple, and user-friendly interface  

---

## ⚙️ Technology Stack

**HTML** – Structured and semantic layout  
**CSS** – Styling with responsive and modern design  
**JavaScript** – Core logic, DOM manipulation, and localStorage  

---

## 🧠 Application Logic Overview

• Uses JavaScript arrays and objects to manage menu and order data  
• Menu items are dynamically rendered on the UI  
• Orders are processed in real-time with instant total calculation  
• localStorage ensures data persistence across browser sessions  
• Modular functions handle:
  - Adding menu items  
  - Displaying menu  
  - Processing orders  
  - Updating total bill  

---

## ▶️ How to Run Locally

1. Clone or download the repository  
2. Open `index.html` in any modern web browser  
3. Start adding menu items and placing orders  

No backend or server setup required.
